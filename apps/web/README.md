# CAS Web App

Frontend React/Vite cho giao diện gọi món, tổng quan bàn, thanh toán, in hóa đơn và dashboard quản lý.

```bash
npm install
npm run dev --workspace @cas/web
```

Nếu chạy riêng trong thư mục này:

```bash
npm install
npm run dev
```

API local mặc định: `http://127.0.0.1:4100`, được Vite chuyển tiếp qua `/api`. Có thể đổi bằng `VITE_DEV_API_TARGET` hoặc `VITE_API_BASE_URL` trong `.env`.
